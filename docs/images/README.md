# Documentation Images Asset Registry

This directory contains visual media, diagrams, and assets referenced inside the AVELIS documentation markdown files.

---

## Directory Organization

* **`er-diagram.png`**: The Entity Relationship Diagram showing model associations and field mappings.
* **`screenshots/`**: Directory containing screenshots and screen capture flows of the frontend single-page application.
  * `home.png`: Core portal landing page screenshot.
  * `collections.png`: Book collection browse interface view.
  * `library.png`: Personal library bookshelves view.
  * `dashboard.png`: Analytics metric summaries page.
  * `journal.png`: Reading logs and journal entry logs.

---

## Asset Guidelines

1. **Format:** Use high-contrast PNG format for design diagrams and screenshots.
2. **Quality:** Maintain clean dimensions (recommended width ≥ 1200px) with legible typography labels.
3. **Paths:** Always reference images relative to the markdown file (e.g. `images/er-diagram.png` in `docs/DATABASE.md`, or `docs/images/er-diagram.png` in root `README.md`).
